项目概述

Agent-PDF 是一个基于 AI 的 PDF 文档智能对话平台，用户可以上传 PDF 文档并与文档内容进行智能对话。 系统支持游客体验和注册用户两种模式，实现了完整的用户认证、配额管理、向量检索和 AI 对话功能。

API 接口服务

🔐 认证服务 /api/auth/

端点 方法 描述 依赖
/api/auth/register POST 用户注册 Supabase Auth, Email
/api/auth/login POST 用户登录 Supabase Auth
/api/auth/logout POST 用户登出 Supabase Auth
/api/auth/me GET 获取当前用户 Supabase Auth

📄 PDF 服务 /api/pdfs/

端点 方法 描述 依赖
/api/upload POST 上传 PDF Quota, Blob, Supabase
/api/parse POST 解析 PDF Parser, Pinecone
/api/pdfs/list GET 获取 PDF 列表 Supabase
/api/pdfs/[id] GET / DELETE PDF 详情/删除 Supabase, Pinecone

💬 聊天服务 /api/chat/

端点 方法 描述 依赖
/api/chat POST AI 对话 (SSE 流式) Quota, LangChain, Pinecone, Qwen
