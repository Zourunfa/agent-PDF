📡 API 接口列表

POST
/api/pdf/upload
上传 PDF 文件，验证格式和大小，生成 pdfId
POST
/api/pdf/parse
解析 PDF 内容，提取文本，生成向量嵌入
GET
/api/pdf/status/[pdfId]
查询 PDF 解析状态（PENDING/Parsing/COMPLETED）
POST
/api/chat
基于 PDF 内容的 AI 对话接口
GET
/api/redis-check
Redis 连接健康检查接口
POST
/api/pdf/delete
删除 PDF 文件及相关缓存数据
