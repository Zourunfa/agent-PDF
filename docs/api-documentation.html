POST /api/auth/register 公开
用户注册，创建新账户并发送验证邮件
请求体
{ "email": "user@example.com", "password": "password123", "name": "张三" // 可选 }
响应
{ "success": true, "message": "注册成功", "emailSent": true, "developmentMode": false }
POST /api/auth/login 公开
用户登录，返回访问令牌
请求体
{ "email": "user@example.com", "password": "password123" }
响应
{ "success": true, "message": "登录成功", "user": { "id": "uuid", "email": "user@example.com", "name": "张三", "role": "user", "emailVerified": true } }
POST /api/auth/logout 需要认证
用户登出，清除会话
响应
{ "success": true, "message": "登出成功" }
GET /api/auth/me 需要认证
获取当前登录用户的信息
响应
{ "success": true, "user": { "id": "uuid", "email": "user@example.com", "name": "张三", "role": "user", "avatar": "https://...", "emailVerified": true, "createdAt": "2024-01-01T00:00:00Z" } }
POST /api/auth/verify-email 公开
验证用户邮箱
请求体
{ "token": "verification_token" }
响应
{ "success": true, "message": "邮箱验证成功" }
POST /api/auth/resend-verification 公开
重新发送验证邮件
请求体
{ "email": "user@example.com" }
POST /api/auth/forgot-password 公开
发送密码重置邮件
请求体
{ "email": "user@example.com" }
POST /api/auth/reset-password 公开
使用令牌重置密码
请求体
{ "token": "reset_token", "password": "new_password123" }
PUT /api/user/profile 需要认证
更新用户资料
请求体
{ "name": "新用户名" }
POST /api/user/avatar 需要认证
上传用户头像
请求体
Content-Type: multipart/form-data { "avatar": <File> }
POST /api/user/change-password 需要认证
修改当前用户密码
请求体
{ "currentPassword": "old_password", "newPassword": "new_password" }
GET /api/user/stats 需要认证
获取用户使用统计数据
响应
{ "success": true, "stats": { "totalUploads": 10, "totalChats": 50, "quotaUsed": 5, "quotaLimit": 20 } }
POST /api/upload 游客/用户
上传 PDF 文件（支持游客和登录用户）
请求体
Content-Type: multipart/form-data { "file": <File> }
响应
{ "success": true, "fileId": "pdf_uuid", "filename": "document.pdf", "status": "parsing" }
GET /api/pdfs/list 需要认证
获取用户上传的 PDF 列表
响应
{ "success": true, "pdfs": [ { "id": "uuid", "filename": "document.pdf", "status": "completed", "createdAt": "2024-01-01T00:00:00Z" } ] }
GET /api/pdfs/[id] 需要认证
获取指定 PDF 的详细信息
响应
{ "success": true, "pdf": { "id": "uuid", "filename": "document.pdf", "pageCount": 10, "status": "completed", "content": "parsed text..." } }
DELETE /api/pdfs/[id] 需要认证
删除指定的 PDF 文件
GET /api/pdfs/[id]/conversations 需要认证
获取指定 PDF 的对话历史
响应
{ "success": true, "conversations": [ { "id": "conv_uuid", "messages": [ {"role": "user", "content": "问题"}, {"role": "assistant", "content": "回答"} ] } ] }
POST /api/manual-text 需要认证
手动输入文本创建临时文档（用于聊天）
请求体
{ "text": "要分析的文本内容...", "title": "文档标题" }
POST /api/chat 游客/用户
与 AI 进行对话（SSE 流式响应）
请求体
{ "message": "用户的问题...", "pdfId": "pdf_uuid", // 可选 "conversationId": "conv_uuid", // 可选 "model": "claude-3-5-sonnet-20241022" // 可选 }
响应 (SSE 流)
event: message data: {"type":"content","text":"部分回答..."} event: message data: {"type":"usage","inputTokens":100,"outputTokens":200} event: done data: {"type":"end"}
GET /api/pdfs/[id]/conversations 需要认证
获取文件的所有对话记录
POST /api/admin/login 管理员
管理员登录（需要 admin 角色）
请求体
{ "email": "admin@example.com", "password": "admin_password" }
GET /api/admin/users 管理员
获取所有用户列表（支持分页）
查询参数
?page=1&limit=20&search=email@example.com
响应
{ "success": true, "users": [/* 用户列表 */], "pagination": { "page": 1, "limit": 20, "total": 100 } }
GET /api/admin/users/[id] 管理员
获取指定用户的详细信息
PATCH /api/admin/users/[id] 管理员
更新用户信息（角色、状态等）
请求体
{ "name": "新用户名", // 可选 "role": "premium", // 可选 "status": "suspended" // 可选 }
DELETE /api/admin/users/[id] 管理员
删除指定用户（包括所有相关数据）
响应
{ "success": true, "message": "用户已成功删除" }
GET /api/quota/stats 需要认证
获取配额使用统计
响应
{ "success": true, "quota": { "limit": 20, "used": 5, "remaining": 15 }, "daily": { "limit": 50, "used": 10 } }
GET /api/guest/quota 游客
检查游客使用配额
响应
{ "success": true, "quota": { "limit": 3, "used": 1, "remaining": 2 } }
POST /api/guest/track 游客
记录游客使用情况（内部调用）
POST /api/guest/migrate 需要认证
游客数据迁移到注册用户
请求体
{ "guestId": "guest_identifier" }

❌ 错误码参考

错误码 HTTP 状态 描述
VALIDATION_ERROR 400 请求参数验证失败
INVALID_EMAIL 400 邮箱格式不正确
WEAK_PASSWORD 400 密码强度不足（至少8位）
INVALID_TOKEN 400 无效的验证令牌
TOKEN_EXPIRED 400 令牌已过期
UNAUTHORIZED 401 未登录或认证失败
USER_DELETED 401 用户已被删除
USER_NOT_FOUND 401 用户不存在或已被封禁
USER_SUSPENDED 403 用户已被封禁
FORBIDDEN 403 无权限访问
QUOTA_EXCEEDED 403 配额已用完
ADMIN_REQUIRED 403 需要管理员权限
NOT_FOUND 404 资源不存在
FILE_TOO_LARGE 413 文件过大（最大10MB）
UNSUPPORTED_FILE_TYPE 415 不支持的文件类型
INTERNAL_ERROR 500 服务器内部错误
DATABASE_ERROR 500 数据库错误
STORAGE_ERROR 500 存储服务错误
AI_SERVICE_ERROR 500 AI 服务错误

📤 标准错误响应格式

{ "success": false, "error": "ERROR_CODE", "message": "用户友好的错误描述", "details": { // 可选 "field": "具体错误字段" }, "timestamp": "2024-01-01T00:00:00Z" }

✅ 标准成功响应格式

{ "success": true, "data": { // 具体数据}, "message": "操作成功", // 可选 "timestamp": "2024-01-01T00:00:00Z" }
