## 🐛 故障排查

### 问题 1: PDF 列表为空
**可能原因**:
- [ ] 没有上传 PDF
- [ ] 上传失败
- [ ] 数据库连接失败

**解决方案**:
1. 检查浏览器控制台是否有错误
2. 检查 Network 标签页的 `/api/upload` 请求
3. 检查 Supabase 仪表板中的 `user_pdfs` 表

### 问题 2: 对话历史为空
**可能原因**:
- [ ] 没有发送消息
- [ ] 消息保存失败
- [ ] 数据库连接失败

**解决方案**:
1. 检查浏览器控制台是否有错误
2. 检查 Network 标签页的 `/api/chat` 请求
3. 检查 Supabase 仪表板中的 `conversation_messages` 表

### 问题 3: API 返回 401 错误
**可能原因**:
- [ ] 用户未登录
- [ ] 认证令牌过期
- [ ] 认证令牌无效

**解决方案**:
1. 检查是否已登录
2. 尝试重新登录
3. 检查浏览器 localStorage 中的认证令牌

### 问题 4: API 返回 403 错误
**可能原因**:
- [ ] 用户不拥有该 PDF
- [ ] 权限验证失败

**解决方案**:
1. 检查 PDF 是否属于当前用户
2. 检查 RLS 策略是否正确配置

### 问题 5: API 返回 500 错误
**可能原因**:
- [ ] 服务器错误
- [ ] 数据库错误
- [ ] 代码异常

**解决方案**:
1. 检查服务器日志
2. 检查数据库连接
3. 尝试重启服务器
