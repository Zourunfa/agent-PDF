## 📊 验证数据

### 验证 PDF 列表数据
在浏览器控制台执行：
```javascript
fetch('/api/pdfs/list').then(r => r.json()).then(d => console.log(d))
```

**应该看到**:
```json
{
  "success": true,
  "data": {
    "total": X,
    "pdfs": [
      {
        "id": "...",
        "filename": "...",
        "fileSize": X,
        "pageCount": X,
        "parseStatus": "completed",
        "uploadedAt": "...",
        "conversationCount": X,
        "lastConversationAt": "..."
      }
    ]
  }
}
```

### 验证对话历史数据
在浏览器控制台执行：
```javascript
fetch('/api/pdfs/{PDF_ID}/conversations').then(r => r.json()).then(d => console.log(d))
```

**应该看到**:
```json
{
  "success": true,
  "data": {
    "pdfId": "...",
    "filename": "...",
    "pageCount": X,
    "messages": [
      {
        "id": "...",
        "role": "user",
        "content": "...",
        "createdAt": "..."
      },
      {
        "id": "...",
        "role": "assistant",
        "content": "...",
        "createdAt": "...",
        "tokens": X,
        "processingTime": X
      }
    ]
  }
}
```
